Poker Kata - Picking the Right Test

The poker kata is another complex kata, the difficulty is turned up a notch on this one.

Explanation

Given two 5 card poker hands, determine which hand wins. Here is the order of hand types weakest to strongest. [Wikipedia link for poker hands]

  • High card (2 being the lowest card and Ace being the highest)
  • A pair (hand contains two cards with the same number of face, if there is a tie then a kicker card determines the winner)
  • Two pair (kicker for ties)
  • Three of a Kind (kicker for ties)
  • Straight (consecutive cards, eg: 3,4,5,6,7)
  • Flush (all cards have the same suit)
  • Full House (a pair and a three of a kind)
  • Four of a Kind (hand containing four cards with the same number of face)
  • Straight Flush (straight where all the cards have the same suit)

Tips!

Think about which tests you write first. Do you start with comparing only hands that contain one card? Or do you try to compare full hands from the beginning? Do you introduce suits right away? If not when is the right time to introduce the concept of a suit? Keep your inputs simple (strings may be good enough). Don't worry about invalid input or invalid hands.
